Rotation placement called "Yosaku" (与作). Be careful of the conspiracy when performing with the L-piece.
Rotation placement called "Hentai-ire" (変態入れ - pervert placement). Opportunities to use it are rare unless you misplace a J or L piece.

Synchro Pass / Frame Placement

"Synchro Pass" (シンクロ渡し - synchro watashi)
Officially called "Frame Placement" (フレーム入れ - frame ire), it uses the Synchro introduced in Chapter 7 to move one column horizontally in mid-air.
Methods include timing the button press visually as it reaches the intended location while moving with DAS charge (visual timing - 目押し me-oshi), or bumping against the wall (kon-kon) and then inputting lever and rotation simultaneously.
S-piece Synchro Pass.
Moves the block horizontally after floating it mid-air using a wall kick.
Synchro Pass is also possible with pieces like T and L.
Opportunities to use it are almost non-existent; rather, when optimizing controls for time attack, be careful as trying to flip normally might unintentionally trigger it.
"Synchro Stand" (シンクロ立て - synchro date) via simultaneous input.
Using this allows performing the "Reverse Somersault" (逆上がり - saka-agari) without charging DAS.

Other Advanced Techniques

"Instant 3-Rotation" (瞬間３回転 - shunkan 3-kaiten)
By using A and C together, you can quickly input the second rotation during horizontal movement without charge delay.
Besides using it for the 3-Rotation Pass, it can also be used when placing the L-piece on the right as shown. Since you must return the lever to neutral for the final rotation, doing it quickly requires considerable practice.
A combination of Instant 3-Rotation, Synchro Stand, and Shift Placement (ずらし入れ - zurashi-ire).
While it was stated that "horizontal and rotation" can be done simultaneously in one frame, "right rotation and left rotation" can also be done simultaneously in one frame.
Since rotation always happens before the drop, this is only meaningful if the first rotation allows a wall kick.
In the left diagram, inputting "IRS-C, Right, BA+Down" allows locking in the same time as "horizontal 1", where a bump (kon-kon) would normally be required.
