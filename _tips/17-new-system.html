Correction of the I and J

For more information see KAN's pages. (Japanese)
New and improved wall and floorkicking behaviour of the I block will make building with them a whole lot easier.

HOLD

For more information see 613's NEXT and HOLD pages. (Japanese)
You should use and abuse the HOLD function in any way that you can, it's especially useful at higher speeds.

3 NEXT blocks

I think the most important addition in Ti is the visibility of 3 NEXT blocks.
You'll adapt quickly from only seeing one block into the future.

The 3 NEXT blocks make it easier to plan out the blocks you'll be getting and how to build them. Especially in combination with the HOLD function.
