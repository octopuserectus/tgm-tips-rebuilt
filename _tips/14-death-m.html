Synchronised Rotation/Movement

Synchronised rotation.
This has been seen in lesson 7.
The same principle applies to the S.
As well as the L and T.
But these two are very hard to use.
Putting a J upright can be done too with this trick.
Using DAS and IRS you can put the L like this, the stick needs to be neutral at the right time.
This requires a lot of practise.
This can be done too by using the synchronous movement trick.
IRS rotation is done before the fall and therefore doesn't cause wallkicks.
